<div id="ppt" markdown="1">

not을 사용하면 반대 답변을 할 수 있어요.

```python
>>> 1==1
True

>>> not 1==1
False
```
참(True)인 표현식은 거짓(False)가 될 수 있어요.

```python
>>> not True
False
```
