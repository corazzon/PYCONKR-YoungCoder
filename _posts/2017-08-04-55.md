Print the names of all the files in a directory:

```python
>>> import os
>>> for file in os.listdir("/home/pi"):
        print(file)
```

Open a web page and read it:

```python
>>> import urllib.request
>>> myurl = "http://www.google.com"
>>> data = urllib.request.urlopen(myurl).read()
>>> print(data)
```
