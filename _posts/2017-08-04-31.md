인덱스는 리스트에서 항목이 어디에 있는지를 말합니다.
```python
>>> fruit = ["apple", “banana", "grape"]
>>> fruit[0]
'apple'
```

파이썬은 항상 0부터 시작하게 됩니다. 
```
['apple', 'banana', 'grape']
    0         1         2
```


</div>

<div id="desc" markdown="1">

좋아하는 3가지 색을 담는 리스트를 만들어보세요.
```python
>>> colors = ["red", "orange", "purple"]
```

인덱스를 사용하여 좋아하는 색의 이름을 출력해봐요.
```python
>>> print(colors[1])
```
