"if 구문"을 다루는 한 가지 방법을 따라가 봅니다. 
{:.summary}

<div id="ppt" markdown="1">
결정이 필요할 때:

"***If*** 배가 고프면, 점심을 먹어요."

"***If*** 휴지통이 꽉 찼으면, 비우러 가요."


조건이 충족되었을 때, <br/>
행동을 취합니다:

```python
>>> name = "Katie"
>>> if name == "Katie":
        print("Hi Katie!")
```

```
Hi Katie!
```
