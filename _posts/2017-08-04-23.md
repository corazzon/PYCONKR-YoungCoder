인터프리터에 부울 값을 입력하면 무슨 일이 일어나게 될까요?

```
>>> True
>>> False
```
'True'와 'False'라는 단어가 대문자로 시작하면 Python은 문자열이나 정수 대신 부울을 처리하는 것을 알고 있죠.

```python
>>> true
>>> false
>>> type(True)
>>> type(“True”)
```
