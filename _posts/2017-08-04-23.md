What happens when we type Boolean values in the interpreter?

```
>>> True
>>> False
```
When the words ‘True’ and ‘False’ begin with upper case letters, Python knows to treat them like Booleans instead of strings or integers.

```python
>>> true
>>> false
>>> type(True)
>>> type(“True”)
```
