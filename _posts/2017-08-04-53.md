* 다양한 선택사항 추가:

"***If*** 민트아이스크림이 있고, 스쿱이 있어요.<br/>
    Or ***else*** 바닐라가 있으면 2개를 먹을래요!<br/>
    Or ***else*** 초콜릿이 있다면 3개를 주세요!<br/>
    Or 아니면 그냥 도넛을 먹을래요."

* elif 절을 사용하여 코드에 더 많은 선택 사항 추가 :

```python
>>> if name == "Katie":
        print("Hi Katie!")
   elif name == "Barbara":
        print("Hi Barbara!")
   else:
        print("Who are you?")
```
