인터프리터에 밑의 코드 중 몇 가지를 시도해봐요.

```python
>>> True and True
>>> False and True
>>> 1 == 1 and 2 == 1
>>> "test" == "test" 
```

곧바로 파이썬이 대답할지 예측할 수 있는지도 확인해 봅시다.
```python
>>> 1 == 1 or 2 != 1
>>> True and 1 == 1
>>> False and 0 != 0
>>> True or 1 == 1
>>> "test" == "tests"
>>> 1 != 0 and 2 == 1
```
