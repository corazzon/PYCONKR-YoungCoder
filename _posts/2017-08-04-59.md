`조건 루프`는 무언가가 발생할 때까지 (또는 어떤 조건이 참일 때까지) 반복됩니다.


```python
>>> count = 0
>>> while (count < 4):
        print("The count is:", count)
        count = count + 1
```

```
The count is: 0
The count is: 1
The count is: 2
The count is: 3
```

while 키워드는 이러한 종류의 루프를 만드는 데 사용되며,
그래서 보통은 while 루프라고 부릅니다.
